Installation & Setup

Step 1: Install the Library 

npm install @synerity/charts

Add the chart library to your Angular project dependencies.

Step 2: Import the Component 

import { SankeyChartComponent } from '@synerity/charts';

@NgModule({
  imports: [SankeyChartComponent],
  // ...
})
export class AppModule { }

Import the SankeyChartComponent in your module or use it as a standalone component.

Step 3: Prepare Your Data 

const sankeyData = {
  nodes: [
    { id: 'A', name: 'Source A' },
    { id: 'B', name: 'Target B' },
    { id: 'C', name: 'Target C' }
  ],
  links: [
    { source: 'A', target: 'B', value: 100 },
    { source: 'A', target: 'C', value: 50 }
  ]
};

Format your data with nodes (sources/targets) and links (flows) between them.

Implementation Examples

Basic Implementation 

// Component TypeScript
export class MyComponent {
  sankeyConfig: SankeyChartConfig = {
    data: {
      nodes: [
        { id: 'Production', name: 'Production' },
        { id: 'Distribution', name: 'Distribution' },
        { id: 'Retail', name: 'Retail' }
      ],
      links: [
        { source: 'Production', target: 'Distribution', value: 1000 },
        { source: 'Distribution', target: 'Retail', value: 800 }
      ]
    },
    width: 600,
    height: 400
  };
}

// Template HTML
<app-sankey-chart 
  [config]="sankeyConfig"
  title="Supply Chain Flow"
  description="Flow of goods through supply chain">
</app-sankey-chart>

Advanced Implementation 

// Component TypeScript
export class MyComponent {
  sankeyConfig: SankeyChartConfig = {
    data: this.generateSankeyData(),
    width: 800,
    height: 500,
    animate: true,
    showValues: true,
    nodePadding: 20,
    linkOpacity: 0.7,
    nodeWidth: 20
  };

  onNodeClick(event: any) {
    console.log('Node clicked:', event);
  }

  onLinkClick(event: any) {
    console.log('Link clicked:', event);
  }
}

Best Practices & Tips

Data Structure

Ensure your nodes have unique IDs and links reference valid source and target nodes.

Flow Visualization

Use meaningful node names and appropriate flow values to clearly represent your data relationships.

Performance Optimization

Limit the number of nodes and links for complex flows to maintain smooth performance.

Accessibility

Provide meaningful titles, descriptions, and tooltips for screen readers. Ensure sufficient color contrast.

Error Handling

Always handle potential errors gracefully, such as missing nodes, invalid links, or empty data.

Interactive Features

Leverage hover effects and click handlers to provide additional context and user interaction.
