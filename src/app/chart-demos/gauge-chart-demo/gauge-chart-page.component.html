API Reference

Configuration Interface 

interface GaugeChartConfig {
  value: number;
  min?: number;
  max?: number;
  width?: number;
  height?: number;
  animate?: boolean;
  colors?: string[];
  label?: string;
  unit?: string;
}

Data Structure 

interface GaugeData {
  value: number;
  min: number;
  max: number;
  label?: string;
  unit?: string;
}

Properties Reference

Property Type Default Description
value number Required Current value to display on the gauge
min number 0 Minimum value for the gauge
max number 100 Maximum value for the gauge
animate boolean true Enable or disable gauge animations
label string '' Label to display on the gauge
