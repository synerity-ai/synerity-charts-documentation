API Reference

Configuration Interface 

interface AreaChartConfig {
  data: AreaData[];
  width?: number;
  height?: number;
  animate?: boolean;
  showGrid?: boolean;
  showAxes?: boolean;
  gradient?: boolean;
  colors?: string[];
  xAxisLabel?: string;
  yAxisLabel?: string;
}

Data Structure 

interface AreaData {
  x: number | string;
  y: number;
  label?: string;
  color?: string;
}

Properties Reference

Property Type Default Description
data AreaData[] Required Array of data points for the area chart
width number 600 Width of the chart in pixels
height number 400 Height of the chart in pixels
animate boolean true Enable or disable chart animations
gradient boolean true Enable gradient fill for the area

Installation & Setup

Step 1: Install the Library 

npm install @synerity/charts

Add the chart library to your Angular project dependencies.

Step 2: Import the Component 

import { AreaChartComponent } from '@synerity/charts';

@NgModule({
  imports: [AreaChartComponent],
  // ...
})
export class AppModule { }

Import the AreaChartComponent in your module or use it as a standalone component.

Step 3: Prepare Your Data 

const areaData = [
  { x: 0, y: 10 },
  { x: 1, y: 15 },
  { x: 2, y: 12 },
  { x: 3, y: 20 },
  { x: 4, y: 18 }
];

Format your data as an array of objects with x and y coordinate properties.

Implementation Examples

Basic Implementation 

// Component TypeScript
export class MyComponent {
  areaConfig: AreaChartConfig = {
    data: [
      { x: 0, y: 10 },
      { x: 1, y: 15 },
      { x: 2, y: 12 },
      { x: 3, y: 20 }
    ],
    width: 600,
    height: 400
  };
}

// Template HTML
<app-area-chart 
  [config]="areaConfig"
  title="Sales Trend"
  description="Monthly sales performance">
</app-area-chart>

Advanced Implementation 

// Component TypeScript
export class MyComponent {
  areaConfig: AreaChartConfig = {
    data: this.generateAreaData(),
    width: 800,
    height: 500,
    animate: true,
    showGrid: true,
    showAxes: true,
    gradient: true,
    colors: ['#3B82F6', '#10B981']
  };

  onPointClick(event: any) {
    console.log('Point clicked:', event);
  }

  onAreaHover(event: any) {
    console.log('Area hovered:', event);
  }
}

Best Practices & Tips

Data Preparation

Ensure your data points are properly sorted by x-coordinate and contain valid numeric values.

Gradient Usage

Use gradients to enhance visual appeal and make the area chart more engaging for users.

Performance Optimization

For large datasets, consider data aggregation or sampling to maintain smooth performance.

Accessibility

Provide meaningful titles, descriptions, and tooltips for screen readers. Ensure sufficient color contrast.

Error Handling

Always handle potential errors gracefully, such as empty data, invalid values, or missing coordinates.

Interactive Features

Leverage hover effects and click handlers to provide additional context and user interaction.
