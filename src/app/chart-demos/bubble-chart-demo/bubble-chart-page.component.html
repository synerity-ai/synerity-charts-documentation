API Reference

Configuration Interface 

interface BubbleChartConfig {
  data: BubbleData[];
  width?: number;
  height?: number;
  animate?: boolean;
  showGrid?: boolean;
  showAxes?: boolean;
  colors?: string[];
  xAxisLabel?: string;
  yAxisLabel?: string;
}

Data Structure 

interface BubbleData {
  x: number;
  y: number;
  radius: number;
  label?: string;
  color?: string;
}

Properties Reference

Property Type Default Description
data BubbleData[] Required Array of bubble data points
width number 600 Width of the chart in pixels
height number 400 Height of the chart in pixels
animate boolean true Enable or disable chart animations
showGrid boolean true Show or hide background grid lines

Installation & Setup

Step 1: Install the Library 

npm install @synerity/charts

Add the chart library to your Angular project dependencies.

Step 2: Import the Component 

import { BubbleChartComponent } from '@synerity/charts';

@NgModule({
  imports: [BubbleChartComponent],
  // ...
})
export class AppModule { }

Import the BubbleChartComponent in your module or use it as a standalone component.

Step 3: Prepare Your Data 

const bubbleData = [
  { x: 10, y: 20, radius: 5, label: 'Point A' },
  { x: 15, y: 25, radius: 8, label: 'Point B' },
  { x: 20, y: 15, radius: 3, label: 'Point C' }
];

Format your data as an array of objects with x, y coordinates, radius, and optional label properties.

Implementation Examples

Basic Implementation 

// Component TypeScript
export class MyComponent {
  bubbleConfig: BubbleChartConfig = {
    data: [
      { x: 10, y: 20, radius: 5 },
      { x: 15, y: 25, radius: 8 },
      { x: 20, y: 15, radius: 3 }
    ],
    width: 600,
    height: 400
  };
}

// Template HTML
<app-bubble-chart 
  [config]="bubbleConfig"
  title="Data Distribution"
  description="Bubble chart showing data points">
</app-bubble-chart>

Advanced Implementation 

// Component TypeScript
export class MyComponent {
  bubbleConfig: BubbleChartConfig = {
    data: this.generateBubbleData(),
    width: 800,
    height: 500,
    animate: true,
    showGrid: true,
    showAxes: true,
    colors: ['#3B82F6', '#10B981', '#F59E0B']
  };

  onBubbleClick(event: any) {
    console.log('Bubble clicked:', event);
  }

  onBubbleHover(event: any) {
    console.log('Bubble hovered:', event);
  }
}

Best Practices & Tips

Data Scaling

Ensure your x, y coordinates and radius values are appropriately scaled for your visualization needs.

Color Schemes

Use consistent color schemes that help distinguish between different data categories or groups.

Performance Optimization

For large datasets, consider data aggregation or sampling to maintain smooth performance.

Accessibility

Provide meaningful labels, descriptions, and tooltips for screen readers. Ensure sufficient color contrast.

Error Handling

Always handle potential errors gracefully, such as empty data, invalid coordinates, or missing radius values.

Interactive Features

Leverage hover effects and click handlers to provide additional context and user interaction.
