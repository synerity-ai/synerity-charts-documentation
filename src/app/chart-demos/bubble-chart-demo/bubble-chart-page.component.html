API Reference

Configuration Interface 

interface BubbleChartConfig {
  data: BubbleData[];
  width?: number;
  height?: number;
  animate?: boolean;
  showGrid?: boolean;
  showAxes?: boolean;
  colors?: string[];
  xAxisLabel?: string;
  yAxisLabel?: string;
}

Data Structure 

interface BubbleData {
  x: number;
  y: number;
  radius: number;
  label?: string;
  color?: string;
}

Properties Reference

Property Type Default Description
data BubbleData[] Required Array of bubble data points
width number 600 Width of the chart in pixels
height number 400 Height of the chart in pixels
animate boolean true Enable or disable chart animations
showGrid boolean true Show or hide background grid lines
