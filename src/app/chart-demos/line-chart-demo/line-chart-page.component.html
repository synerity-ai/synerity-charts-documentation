API Reference

Configuration Interface 

interface LineChartConfig {
  data: ChartData[];
  width?: number;
  height?: number;
  animate?: boolean;
  showPoints?: boolean;
  showGrid?: boolean;
  curveType?: 'linear' | 'monotoneX' | 'step' | 'stepAfter' | 
              'stepBefore' | 'basis' | 'cardinal' | 'catmullRom';
  strokeWidth?: number;
  colors?: string[];
}

Data Structure 

interface ChartData {
  label: string;
  value: number;
  color?: string;
}

Properties Reference

Property Type Default Description
data ChartData[] Required Array of data points to display in the chart
width number 600 Width of the chart in pixels
height number 400 Height of the chart in pixels
animate boolean true Enable or disable chart animations
showPoints boolean true Show or hide data points on the line
showGrid boolean true Show or hide background grid lines
curveType string 'linear' Type of curve interpolation to use
strokeWidth number 2 Width of the line stroke in pixels

Usage Examples

Basic Configuration 

const config: LineChartConfig = {
  data: [
    { label: 'Jan', value: 65 },
    { label: 'Feb', value: 78 },
    { label: 'Mar', value: 90 }
  ],
  width: 600,
  height: 400
};

Advanced Configuration 

const config: LineChartConfig = {
  data: [
    { label: 'Jan', value: 120, color: '#3B82F6' },
    { label: 'Feb', value: 180, color: '#10B981' }
  ],
  animate: true,
  showPoints: true,
  showGrid: false,
  curveType: 'monotoneX',
  strokeWidth: 3
};

Installation & Setup

Step 1: Install the Library 

npm install @synerity/charts

Add the chart library to your Angular project dependencies.

Step 2: Import the Component 

import { LineChartComponent } from '@synerity/charts';

@NgModule({
  imports: [LineChartComponent],
  // ...
})
export class AppModule { }

Import the LineChartComponent in your module or use it as a standalone component.

Step 3: Prepare Your Data 

const chartData: ChartData[] = [
  { label: 'January', value: 120, color: '#3B82F6' },
  { label: 'February', value: 180, color: '#10B981' },
  { label: 'March', value: 150, color: '#F59E0B' }
];

Format your data according to the ChartData interface with label, value, and optional color properties.

Best Practices & Tips

Data Preparation

Sort data chronologically for time series and ensure consistent formatting with meaningful labels.

Responsive Design

Use percentage-based widths or responsive containers to ensure your chart adapts to different screen sizes.

Performance Optimization

For large datasets, consider data aggregation or virtual scrolling to maintain smooth performance.

Accessibility

Provide meaningful titles, descriptions, and ARIA labels for screen readers. Ensure sufficient color contrast.

Error Handling

Always handle potential errors gracefully, such as empty data, invalid values, or network issues.

Color Schemes

Use consistent color schemes that match your application's design system and ensure good contrast ratios.
