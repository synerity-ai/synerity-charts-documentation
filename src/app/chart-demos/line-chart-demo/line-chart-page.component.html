Line Chart

Time series visualization with smooth curves and interactive data points

Overview

Line charts are perfect for visualizing time series data and trends over time. They connect data points with lines to show progression, patterns, and relationships in your data.

API Reference

LineChartConfig Interface

interface LineChartConfig {
  data: ChartData[];
  width?: number;
  height?: number;
  animate?: boolean;
  showPoints?: boolean;
  showGrid?: boolean;
  curveType?: 'linear' | 'monotoneX' | 'step' | 'stepAfter' | 
              'stepBefore' | 'basis' | 'cardinal' | 'catmullRom';
  strokeWidth?: number;
  colors?: string[];
}

ChartData Interface

interface ChartData {
  label: string;
  value: number;
  color?: string;
}

Installation

Install the @synerity/charts library:

npm install @synerity/charts

Basic Usage

Vanilla JavaScript

import { LineChart } from '@synerity/charts';

const data = [
  { label: 'Jan', value: 65, color: '#3B82F6' },
  { label: 'Feb', value: 78, color: '#10B981' },
  { label: 'Mar', value: 90, color: '#F59E0B' }
];

const chart = new LineChart({
  container: '#chart-container',
  data: data,
  options: {
    width: 600,
    height: 400,
    animate: true,
    showPoints: true,
    curveType: 'monotoneX'
  }
});

Angular Component

<app-line-chart 
  [config]="lineChartConfig"

Step-by-Step Guide

1

Prepare Your Data

Format your data as an array of objects with label, value, and optional color properties.

const data = [
  { label: 'Q1', value: 100, color: '#3B82F6' },
  { label: 'Q2', value: 150, color: '#10B981' },
  { label: 'Q3', value: 120, color: '#F59E0B' },
  { label: 'Q4', value: 200, color: '#EF4444' }
];
2

Configure Chart Options

Set up the chart configuration with your preferred options.

const config = {
  data: data,
  width: 600,
  height: 400,
  animate: true,
  showPoints: true,
  showGrid: true,
  curveType: 'monotoneX',
  strokeWidth: 2
};
3

Initialize the Chart

Create the chart instance and render it in your container.

const chart = new LineChart({
  container: '#chart-container',
  data: data,
  options: config
});

Best Practices

Data Preparation

  • • Sort data chronologically for time series
  • • Use consistent color schemes
  • • Include meaningful labels
  • • Handle missing data appropriately

Visual Design

  • • Choose appropriate curve types
  • • Use grid lines for readability
  • • Ensure sufficient contrast
  • • Keep animations subtle

Troubleshooting

Chart not rendering?

Check that your container element exists and has proper dimensions.

Data not updating?

Ensure you're calling the update method with new data.

Performance issues?

Consider reducing animation complexity for large datasets.
