Installation & Setup

Step 1: Install the Library 

npm install @synerity/charts

Add the chart library to your Angular project dependencies.

Step 2: Import the Component 

import { HeatmapChartComponent } from '@synerity/charts';

@NgModule({
  imports: [HeatmapChartComponent],
  // ...
})
export class AppModule { }

Import the HeatmapChartComponent in your module or use it as a standalone component.

Step 3: Prepare Your Data 

const heatmapData = [
  { x: 'A', y: 'X', value: 10 },
  { x: 'A', y: 'Y', value: 20 },
  { x: 'B', y: 'X', value: 15 },
  { x: 'B', y: 'Y', value: 25 }
];

Format your data as an array of objects with x, y coordinates and value properties.

Implementation Examples

Basic Implementation 

// Component TypeScript
export class MyComponent {
  heatmapConfig: HeatmapChartConfig = {
    data: [
      { x: 'Q1', y: 'Sales', value: 100 },
      { x: 'Q2', y: 'Sales', value: 150 },
      { x: 'Q1', y: 'Marketing', value: 80 },
      { x: 'Q2', y: 'Marketing', value: 120 }
    ],
    width: 600,
    height: 400
  };
}

// Template HTML
<app-heatmap-chart 
  [config]="heatmapConfig"
  title="Performance Heatmap"
  description="Quarterly performance across departments">
</app-heatmap-chart>

Advanced Implementation 

// Component TypeScript
export class MyComponent {
  heatmapConfig: HeatmapChartConfig = {
    data: this.generateHeatmapData(),
    width: 800,
    height: 500,
    animate: true,
    showValues: true,
    showGrid: true,
    colorScheme: 'RdYlBu',
    cellPadding: 2
  };

  onCellClick(event: any) {
    console.log('Cell clicked:', event);
  }

  onCellHover(event: any) {
    console.log('Cell hovered:', event);
  }
}

Best Practices & Tips

Data Organization

Organize your data logically with meaningful x and y axis labels for better readability and interpretation.

Color Schemes

Choose appropriate color schemes that match your data context and ensure good contrast for accessibility.

Performance Optimization

For large datasets, consider data aggregation or pagination to maintain smooth performance.

Accessibility

Provide meaningful titles, descriptions, and tooltips for screen readers. Ensure sufficient color contrast.

Error Handling

Always handle potential errors gracefully, such as empty data, invalid values, or missing coordinates.

Interactive Features

Leverage hover effects and click handlers to provide additional context and user interaction.
