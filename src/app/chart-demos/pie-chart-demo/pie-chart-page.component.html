API Reference

Configuration Interface 

interface PieChartConfig {
  data: ChartData[];
  width?: number;
  height?: number;
  animate?: boolean;
  showLabels?: boolean;
  showLegend?: boolean;
  innerRadius?: number;
  outerRadius?: number;
  colors?: string[];
}

Data Structure 

interface ChartData {
  label: string;
  value: number;
  color?: string;
}

Properties Reference

Property Type Default Description
data ChartData[] Required Array of data points to display in the chart
width number 600 Width of the chart in pixels
height number 400 Height of the chart in pixels
animate boolean true Enable or disable chart animations
showLabels boolean true Show or hide data labels on slices
showLegend boolean true Show or hide the legend
innerRadius number 0 Inner radius for donut chart (0 for pie chart)
outerRadius number 200 Outer radius of the chart

Usage Examples

Basic Configuration 

const config: PieChartConfig = {
  data: [
    { label: 'Red', value: 30 },
    { label: 'Blue', value: 25 },
    { label: 'Green', value: 45 }
  ],
  width: 600,
  height: 400
};

Advanced Configuration 

const config: PieChartConfig = {
  data: [
    { label: 'Sales', value: 120, color: '#3B82F6' },
    { label: 'Marketing', value: 80, color: '#10B981' }
  ],
  animate: true,
  showLabels: true,
  showLegend: false,
  innerRadius: 50,
  outerRadius: 150
};

How To Use Pie Chart

Complete implementation guide and usage examples

Installation

Install the synerity-charts package using npm.

Basic Usage

Import the PieChartModule in your Angular module.

Component Usage

Use the app-pie-chart component in your template.

Data Example

Provide data in the following format: Array of objects with label and value properties.

Configuration Example

Configure chart properties: width, height, animate, showLabels, showLegend, innerRadius, outerRadius.

Best Practices

Use meaningful labels, limit to 5-7 segments for readability, and ensure proper color contrast.

Troubleshooting

Common issues: data not displaying (check data format), animations not working (ensure animate: true), labels overlapping (adjust innerRadius/outerRadius).
