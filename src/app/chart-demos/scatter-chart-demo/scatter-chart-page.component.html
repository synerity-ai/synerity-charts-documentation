API Reference

Configuration Interface 

interface ScatterChartConfig {
  data: ScatterData[];
  width?: number;
  height?: number;
  animate?: boolean;
  showGrid?: boolean;
  showAxes?: boolean;
  pointSize?: number;
  colors?: string[];
  xAxisLabel?: string;
  yAxisLabel?: string;
}

Data Structure 

interface ScatterData {
  x: number;
  y: number;
  label?: string;
  color?: string;
  size?: number;
}

Properties Reference

Property Type Default Description
data ScatterData[] Required Array of data points to display in the chart
width number 600 Width of the chart in pixels
height number 400 Height of the chart in pixels
animate boolean true Enable or disable chart animations
showGrid boolean true Show or hide background grid lines
showAxes boolean true Show or hide axis lines and labels
pointSize number 6 Size of scatter plot points in pixels
